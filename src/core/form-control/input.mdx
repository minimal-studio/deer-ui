# Input

> 输入验证控件, 基础类提供基本验证

## Usage

<Playground>
  <Input title="输出 number 类型" inputType="number" onChange={e => console.log(e)}/>
  <hr/>
  <Input title="输出字符串类型" onChange={e => console.log(e)}/>
  <hr/>
  <Input title="带有正则过滤器" filter={val => (val + '').replace(/[^0-9]/g, '')} />
  <hr/>
  <Input n="phone" />
  <hr/>
  <Input
    title="带有按钮的输入框"
    inputType="number"
    inputBtnConfig={{
      action: (e) => console.log(e),
      text: '按钮'
    }}/>
</Playground>
