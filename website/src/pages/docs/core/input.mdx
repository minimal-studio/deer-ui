# Input

> 文本输入增强，提供输入验证支持

## Basic Usage

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input
      placeholder="请输入"
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## 固定的 Title

> 为 Input 添加 title，与 placeholder 不同，title 并不会在输入后消失

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="固定 Title" 
      placeholder="placeholder"
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## 输出数字类型

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="输出 number 类型" 
      placeholder="输出数字" 
      outputType="number" 
      propsForInput={{
        type: 'number'
      }}
      onChange={e => console.log(e)}/>
  )
}
render(<Example />);
```

## 过滤器

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input 
      title="只能输入数字" 
      filter={val => (val + '').replace(/[^0-9]/g, '')} />
  )
}
render(<Example />);
```

## With Icon

```jsx live noInline
function Example() {

    <Input n="phone" />

```

## With Button

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <>
      <Input
        title="带按钮的输入框"
        outputType="number"
        inputBtnConfig={{
          action: (e) => console.log(e),
          text: '按钮'
        }}/>
      <hr />
      <Input
        title="带按钮的输入框"
        outputType="number"
        inputBtnConfig={{
          action: (e) => console.log(e),
          icon: 'search',
          text: ''
        }}/>
    </>
  )
}
render(<Example />);
```

## 获取焦点时 Title 上浮

```jsx live noInline
import { Input } from '@deer-ui/core'

function Example() {
  return (
    <Input n="phone" title="电话号码" placeholder="请输入" flowTitle />
  )
}
render(<Example />);
```
